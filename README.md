## RelayScheduler

来週のリレー投稿担当者を順番に選んでSlackに投稿する。週ごとにファイルに保存。指定したチャンネルのメンバーを「Slack IDのsha256」の順で並べる。履歴の最後を取り出して、その次にあたる人から順番。履歴ファイルの番号はユリウス歴1年1月1日（月）を第0週とした週番号である。すでにファイルがある場合はリマインドする。cronで呼び出して使うのが基本。
